Pixel Hopper is an early game development venture, mimicking the core gameplay of Lima Sky's Doodle Jump (2009). Developed in Processing, a Java-based open-source programming language for art and visual media creation, Pixel Hopper takes the concept behind Doodle Jump and brings it into the realm of 16-bit retro arcade-style platformers. The entirety of Pixel Hopper was written in Processing without the use of any third-party plug-ins.

Character, environment, and menu art were created by Brent Lopez.

This game is not intended to be sold.
